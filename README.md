# vue-chat

使用Vue构建一个聊天应用

主要用于学习vue及周边生态工具的用法：
    Vue
    Vuex
    Vue Router
    Vue CLI
    Pusher

推荐安装Chrome/ vue-devtool 去更好的理解state transfer

```
npm install
```
